Перечисления

◆ CoreTaskFlag

enum CoreTaskFlag

#include <sxcore.h>

значения определющие поведение задачи

Элементы перечислений
CoreTF_REPEATING 

Задача будет повторяться, если не указано - выполнится только один раз

CoreTF_THREADSAFE 

Задача может быть выполнена в любом потоке

CoreTF_FRAME_SYNC 

Задаче необходима синхронизация по границе кадра

CoreTF_ON_SYNC 

Это выполняется в момент синхронизации(выполняет необходимые действия для обмена данными во время синхронизации)

CoreTF_SINGLETHREADED 

Задача выполняется в главном потоке, один раз

CoreTF_SINGLETHREADED_REPEATING 

Задача выполняется в главном потоке, повторяется

CoreTF_BACKGROUND 

Задача выполняется в фоне(не ожидает синхронизации) один раз

CoreTF_BACKGROUND_REPEATING 

Задача выполняется в фоне(не ожидает синхронизации), по завершении повторяется

CoreTF_BACKGROUND_SYNC 

Задача может выполняться в любом потоке, ожидает синхронизации, выполняется один раз

CoreTF_BACKGROUND_SYNC_REPEATING 

Задача может выполняться в любом потоке, ожидает синхронизации, выполняется многократно
