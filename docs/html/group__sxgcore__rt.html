Аргументы
widthширина
heightвысота
levelsколичество mip-map уровней
usageпризнак применения, возможные значения из D3DUSAGE_
formatформат из D3DFORMAT
poolгде будет размещена текстура, из D3DPOOL
nameимя rt
coeffullscreenкоэфициент размеров rt относительно области рендера (если rt и размер области одинаковы то 1, если rt меньша на 0.5 то 0.5) если указаны фиксированные значения то ставить 0, это нужно для устновления размеров при восстановлении устройства
